Operators in Python

What is an Operator?

In Python, an operator is one or more symbols used to perform actions on values. Some actions can be defining or causing change to values.

Different Types of Operators

Arithmetic Operators

These operators change values of given numbers or variables by adding(+), subtracting(-), multiplying(*), and dividing(/).

Comparison Operator

These operators compair values of given numbers or variables by using equal to (=), less than(<), greater than(>), equal to or less than(<=), more than or equal to(>=), and not equal to(!=).

Assignment Operator

These operators are very similar to Arithmetic Operators but will assign values to variables instead using equal to (=), adding to the value (+=), subtracting from it (-=), multiplying the variable (*=), and dividing the number (/=)

Logical Operator

These operators use logic and then gives back a true or false respond depending if the statement you made is correct or not. It uses "and," "or," and "not"
